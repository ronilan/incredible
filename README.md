# Incredible

Incredible is a new **Experimental**, **Modular**, **Declarative**, **Event Driven**, **Reactive**, Rust Text User Interface (TUI) framework for building cross-platform apps. 

Incredible Apps can be built for the **terminal** (cross platform), a standalone **native window** (Mac OS & Windows 11), or the **browser** (via WASM) using one unified code base.

The framework is built from the ground up following basic UI principles. It implements a core element tree structure, enables the composition of new elements from encapsulated and/or fully reusable sub elements, and provides out of the box support for most modern terminal capabilities.

## Status

Incredible is currently in **private alpha**.
